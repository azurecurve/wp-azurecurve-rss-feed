== Description ==
Provides opposite rss feed to that configured in WordPress; e.g. if WordPress is configured for summary then an alternative feed called detail will be created, or if WordPress is configured for a detailed feed then an alternative feed called summary is created.

The feed can be access via one of the following paths (depending on which one is created):
http://yourdomain/feed/detail
http://yourdomain/feed/summary

== Installation ==
To install the plugin copy the <em>azurcurve-rss-feed</em> folder into your plug-in directory and activate it.
